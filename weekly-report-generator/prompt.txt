## 01  magic-resume

`Magic Resume`是一款现代化的在线 AI 简历编辑器，主打隐私安全与AI 辅助。它基于 TanStack Start、TypeScript 等现代技术栈构建，提供了所见即所得的编辑体验。最大的亮点在于：**无需注册、无需登录，所有简历数据默认存储在本地浏览器中**，彻底杜绝数据上云带来的隐私担忧。

![](https://image01-1315567783.cos.ap-guangzhou.myqcloud.com/mdnice/2026/04/20260408155844039.webp)

- 自动保存与智能一页适配，编辑不丢内容，自动优化排版让简历刚好一页。

## 快速安装与使用

### 在线直接使用（推荐）

直接访问官网 `magicv.art`，打开即可编辑，不用注册、不用安装、不用登录。

### 本地部署运行
1. 克隆项目
```
git clone https://github.com/JOYCEQL/magic-resume.git
cd magic-resume
```
2. 安装依赖
```
pnpm install
```
3. 启动开发服务
```
pnpm dev
```
4. 浏览器访问 http://localhost:3000 即可使用。

### Docker部署
1. 安装Docker与Docker Compose
2. 项目根目录执行
```
docker compose up -d
```
自动构建镜像并后台启动，适合个人服务器自托管。
