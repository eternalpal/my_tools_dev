FFmpeg GIF 命令生成器

📊 原始文件分析：
📄 文件名: -
📏 尺寸: -
⏱️ 原帧率: - FPS
🎞️ 总帧数: -
生成的 FFmpeg 命令：

ffmpeg -i "input.gif" -vf "fps=10,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse" -y "output_opt.gif"

📖 快速使用指南

1 下载并准备 FFmpeg
从 此处下载 FFmpeg 压缩包。解压后，找到 ffmpeg.exe 文件，将其复制到存放 GIF 图片的同一个文件夹中。
2 打开终端 (CMD)
在存放图片的文件夹路径栏输入 cmd 并回车，或者在文件夹内按住 Shift + 右键 选择“在此处打开 PowerShell 窗口”。
3 粘贴命令运行
复制上方生成的命令，在打开的黑色窗口中点击鼠标右键完成粘贴，然后按下回车键 (Enter)。等待几秒，优化后的 GIF 就生成在同文件夹下了。
